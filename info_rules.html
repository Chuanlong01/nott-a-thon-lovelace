  2. Participants must register using the signup link by 10th February 2022.
  3. Participation fee is RM 30 per group. Whether there are only 1 or 3 members in a group, the fee is fixed.

Platform and Timeline

  1. The competition will be hosted on HackerRank.
  2. The competition will run for 2 hours, from 1500hrs to 1700hrs on 30th January 2021 (Saturday).
  3. Submissions outside of the competition time will not be considered.
  4. Each group must create a HackerRank account which will be used for code submission.
  5. Only ONE submission account per team. Submissions on multiple accounts will NOT be consolidated.
  6. Groups must submit the HackerRank username of the representative account when registering.
  7. Accounts that do not match the submitted username will not be considered.
  8. Programing languages allowed:
    1. C
    2. C++
    3. Java
    4. Java 8
    5. Kotlin
    6. Python 2
    7. Python 3
  9. Run time limits for each language can be found at this link. https://www.hackerrank.com/environment/languages
  10. Participants are expected to familiarize themselves with the platform beforehand.
  11. Participants are allowed to use standard libraries that are available in HackerRank. You can refer to the following link.

https://www.hackerrank.com/environment/languages

*NOTE: Library list for Python 2 & Python 3, IS INCORRECT. ONLY PYTHON STANDARD LIBRARIES ARE INSTALLED. Refer to https://docs.python.org/3/library/

 

Foul Play

  1. Groups found to have violated the Foul Play Clauses in part 6 will be disqualified.
  2. Please contact efyjl10@nottingham.edu.my  to report any foul play.
  3. The organizers reserve the right to disqualify any group for unacceptable conduct or violates any of the rules stipulated in this document.

 

Technical Support

  1. Before the competition, please email your queries (if any) to efyjl10@nottingham.edu.my 
  2. During the competition, you can contact the organizers on Discord, via this link https://discord.gg/KpGVGqbyQg 
  3. Do not share your codes or solutions in the competition Discord server
  4. No compensation/refunds/time advantage/point advantage will be given for any technical issues.
  5. Please ensure you have a stable internet connection for the entire duration of the competition.

 

Questions

  1. There are seven challenges / questions throughout competition
    1. 3 Easy questions worth 10 points each.
    2. 3 Medium questions worth 20 points each.
    3. 2 Hard questions worth 30 points each.
  2. 5 test cases are set for easy and medium questions.
  3. 10 test cases are set for hard questions.
  4. Participants are free to choose the sequence of answering.
  5. Successfully answering each test case will grant points.
  6. Please refer to part m. for the run time duration for each programing language.

s-CPD Points 

  1. 4 s-CPD points are claimable for this event for applicable students from University of Nottingham Malaysia.
  2. Ensure your Library-ID provided during registration is accurate.
